# Spring AI Agent Workshop

Welcome to the Spring AI Agent Workshop! This repository contains a series of hands-on steps to help you master building AI-powered applications using Spring Boot and Spring AI.

## Workshop Structure

*   **[Step 01 - Basics](./step-01)**: Introduction to Spring AI and setting up your first AI interaction.
*   **[Step 02 - Chatbot](./step-02)**: Building a real-time interactive chatbot using WebSockets and Ollama.

## Prerequisites

*   **Java 24+**
*   **Maven**
*   **Ollama**: Ensure you have Ollama installed and running locally.
    *   Pull the model: `ollama pull llama3.2`

## Getting Started

Navigate to each step's directory and follow the specific `README.md` instructions to run the application.

```bash
cd step-02
mvn spring-boot:run
```

Happy Coding!
